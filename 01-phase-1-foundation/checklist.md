# Phase 1 Foundation Checklist

Use this checklist to verify that every team member has a working development environment.

## Setup Tasks

- [ ] Docker Desktop installed and running
- [ ] Clone dashboard repo (fresh, not forked)
- [ ] Run docker compose command from README
- [ ] Access frontend at localhost:5173
- [ ] Access backend at localhost:8000
- [ ] Login with seeded admin credentials
- [ ] Click through UI - see training modules, FAQ
- [ ] Every team member can run it on their machine

## Verification

Once all checkboxes are complete, you should be able to:

- Open the dashboard in your browser
- Navigate through the training modules
- View FAQ content
- See the chatbot interface
- Make changes to code and see them reflected in the running application

## Next Steps

Once everyone on the team has completed this checklist, you're ready to move on to Phase 2 and start customizing the dashboard for your organization.
