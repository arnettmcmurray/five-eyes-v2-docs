## Your Mission

1. Make sure everyone has what they need
2. Create architecture showing how v1 + v2 fit together
3. Plan Week 2 sprint with clear tasks

---

## Tasks

### 1. Facilitate Team Coordination (All Week)

**What to do:**

- Run standups Wednesday & Friday
- Check in with team members daily
- Unblock people who are stuck
- Connect teammates who need to sync (FE ↔ BE)

**Deliverable:** Team stays moving, no one is blocked

---

### 2. Create Architecture Diagram (Wednesday/Thursday)

**What to do:**

- Show how AI components integrate with v1
- Include:
  - Frontend (React components)
  - Backend (FastAPI endpoints)
  - Database (new tables)
  - LLM provider (external API)
  - Data flow (arrows showing requests)

**Tools:** draw.io, Excalidraw, or any diagramming tool

**Deliverable:** Architecture diagram (image or link)

---

### 3. Document v1 + v2 Integration Points (Thursday)

**What to do:**

- List where v2 touches v1:
  - Which v1 routes do we extend?
  - Which v1 tables do we reference?
  - How does auth work?
- Identify potential conflicts or risks

**Deliverable:** Integration points doc (markdown)

---

### 4. Create Week 2 Sprint Plan (Thursday/Friday)

**What to do:**

- Based on everyone's Week 1 deliverables, break Week 2 into tasks:
  - Frontend: Build chat widget
  - Backend: Implement API endpoint
  - Cybersecurity: Test guardrails
  - Integration: Connect pieces

**For each task:**

- Clear description
- Owner
- Acceptance criteria
- Estimated time

**Deliverable:** Week 2 sprint plan (markdown)

---

### 5. Identify Risks (Friday)

**What to do:**

- List what could go wrong:
  - LLM costs too much?
  - Integration breaks v1?
  - Team member unavailable?
- For each risk, note mitigation strategy

**Deliverable:** Risk register (table or list)
