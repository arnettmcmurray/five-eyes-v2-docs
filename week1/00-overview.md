## The Project (30-second version)

We're adding an **AI chat assistant** to Five Eyes' cyber training platform.

**It will:**

- Explain security concepts in plain language
- Answer user questions (but NOT give away quiz answers)
- Help people actually learn instead of just clicking through

**It won't:**

- Spy on individual users
- Give away correct answers
- Replace the existing training
